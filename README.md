# Power BI Design & DAX Exploration

## Project Objective
This project was focused on mastering the **User Interface (UI)** and **Technical Design** capabilities of Power BI Desktop. The primary goal was to move beyond default settings and experiment with advanced formatting, DAX table manipulation, and professional report aesthetics.

## Key Features & Learning Milestones

### Technical Implementation
* **DAX Table Manipulation:** Created a new calculated table using **DAX** to restructure the data, allowing for more specialized and effective graphing.
* **Conditional Formatting:** Experimented with logic-based styling within tables to improve scannability and emphasize specific data trends.
* **Core Functionality:** Integrated a variety of visuals including Bar Charts, KPI Cards, Slicers, and interactive Tables.

### UI/UX & Visual Design
* **Dark Theme Experimentation:** Developed the report using a "Dark Mode" aesthetic to test visual contrast and readability.
* **Gradient Backgrounds:** Applied custom gradient backgrounds to enhance the professional "Executive Dashboard" feel.
* **Design Layout:** Focused on clean alignment and modern design principles to create an intuitive user experience.
